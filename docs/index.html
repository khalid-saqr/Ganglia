The idea

Wittgenstein for the developer seeing Ganglia first

In Philosophical Investigations, Wittgenstein uses language-games to show that meaning is not an isolated label attached to a word. Meaning depends on the activity, rules, and situation in which language is used.

01

Language is an activity

A phrase such as a request, command, comparison, classification, or warning only becomes legible inside the activity where it is used.

02

Rules shape meaning

Language-games are not rigid programming languages. They are practical patterns with enough rules to make a move count as correct, broken, missing, or repaired.

03

Games clarify use

Ganglia treats an LLM response as a move in a designed language-game: the operator names the task, the schema defines the move, and validators decide whether it passes.

LLM reasoning design

From free-answering to observable reasoning products

Ganglia does not claim to control hidden model cognition. It controls the observable product: the structured reasoning object the model must emit and pass through validation.

1. OperatorChoose the language-game.
2. Prompt contractCompile the instructions.
3. JSON schemaForce a checkable shape.
4. Semantic checksValidate the move, not only the syntax.
5. Repair loopAsk for a corrected move before release.
6. TraceKeep the observable record.

Built-in games

Three ways to make a model play by rules

coordinate_game

Forces abstract entities into a two-axis 0-10 coordinate system.

  • mapping
  • plotting
  • positioning

grid_game

Forces content into a comparative matrix.

  • comparison
  • classification
  • MECE table forcing

adversarial_grid_chain

Applies adversarial testing, then formalises the result as a failure matrix.

  • stress-testing
  • falsification
  • risk detection

Reference-grounded claims

Sources for the philosophical translation

The site uses Wittgenstein's language-game idea as a design lens, not as a claim that LLM cognition is fully controllable. The project controls the response artifact that can be validated, repaired, traced, and returned.

  • Stanford Encyclopedia of Philosophy: Ludwig Wittgenstein — overview of language-games, forms of life, and rule-governed language use.
  • Internet Archive: Philosophical Investigations — bibliographic reference for Wittgenstein's original work in which language-games are developed.
  • Ganglia README — installation, API, operators, traces, configuration, and production notes.
